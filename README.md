# Internship-at-PLN-Nusantara-Power-UPDK-Bandar-Lampung
This repository contains an internship project at PT. PLN Nusantara Power UPDK Bandar Lampung, focusing on Social Return on Investment (SROI) analysis to measure the success of the CSR Program “Peduli DAS Sungai V” in the Air Hitam Sub-Watershed.

This project was conducted as a group internship (3 members). In the task distribution, the author was primarily responsible for the analysis of the “Peduli DAS Sungai V” CSR program, while other CSR programs were handled by team members. All analyses were carried out collaboratively.
